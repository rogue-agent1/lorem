# lorem

Generate placeholder text.

One file. Zero deps. Fills space.

## Usage

```bash
python3 lorem.py               # 1 paragraph
python3 lorem.py 3             # 3 paragraphs
python3 lorem.py words 50      # 50 words
python3 lorem.py sentences 10  # 10 sentences
python3 lorem.py chars 280     # exactly 280 characters
python3 lorem.py list 5        # bullet list
python3 lorem.py html 3        # HTML paragraphs
```

## Requirements

Python 3.8+. No dependencies.
